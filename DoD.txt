PANGEA - DEFINITION OF DONE

- Last version of running code deployed on virtual pc
- Last version of source code deployed on virtual pc
- Code execution is tested on virtual pc
- Code deployed on the test environment identical to production environment
- Configurations or build changes documented on Git 
- Running code is complete when it goes through tests
- Running code is complete when technical debt is removed
- Running code is complete when it meets feature’s acceptance criteria
- Unit, integration, regression and performance tests are necessary
- Concern about non-functional requirements is necessary
- Refactoring and continuos improvement of code are necessary
- Pangea app platforms: Android - iOS
